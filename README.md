# AaTherAl Bot

A simple and powerful bot for Discord made using Discord.js

## Usage

### Moderation Commands:

- `ban`: to Ban a member from the server.
- `kick`: to kick a member from the server.

### Helpful Commands:

- `ping`: To check if the bot is working or not.
